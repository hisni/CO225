Compile and run MainServer.java

Connect to the to the server through port no 2000.

Then client required to provide Name and then Symbol of the security he/she is willing to bid on. 
Then client can start Bidding.
The world "quit" can be entered(without quotes) at any point in the program to exit it.

In GUI there are 4 columns which includes symbol, Security Name , Current price and sShow Bid History.
Price column includes current price and it gets updated every 500ms.
Show Bid History Column includes buttons and by Clicking the button can view all valid bids made on that particular symbol in a new window, 
Text area below the table displays all the bid History on all the symbols. It gets updated when a valid bid is made.
